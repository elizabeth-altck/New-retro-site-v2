что такое MVP для нас

MVP GIF WORLD — сайт, который можно отправить ссылкой:

  1. Главная выглядит живой и «официальной»
  2. Минимум 3 настоящие комнаты (main + gallery + diary + zhanna)
  3. Guest book реально работает (FormSubmit или аналог)
  4. Nav не ведёт в пустые 202 без предупреждения
  5. Деплой + smoke test (index → main → комнаты → 404)

что сознательно НЕ в MVP

  • Ramen Explorer / рабочий стол
  • Полный scrapbook-редизайн
  • Retro photos в gallery (отложено)
  • Tamagotchi с сытостью в LIVE SYSTEM
  • Rose cursor (parked в отдельных файлах)

как смотреть на телефоне / планшете
  1. Лучший способ — ссылка в интернете. Файл лежит на сайте как roadmap.html. После деплоя открой на любом устройстве: твой-сайт.ru/roadmap.html — телефон, планшет, чужой ноутбук, всё одинаково.
  2. Без деплоя, за 2 минуты: зайди на app.netlify.com/drop → перетащи файл roadmap.html → получишь временную ссылку → открой её на телефоне.
  3. Только на своём iPhone: AirDrop файла → «Файлы» → нажми на файл → откроется в Safari. На Android: отправь себе в Telegram → скачай → «Открыть в Chrome».

Локальный файл на компе в браузере — ок. На телефоне без ссылки — неудобно; нужен URL или AirDrop.
