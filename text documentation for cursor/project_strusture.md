Содержит:

* список страниц:
  - Главная
  - Галерея
  - Музыкальная комната
  - Рабочий стол
  - Дом / комнаты
  - Архив
  - Секретные страницы

* способы навигации:
  - меню
  - интерактивные объекты (GIF, картинки, двери, окна)
  - скрытые переходы
  - нестандартные элементы интерфейса

Изменяется по мере роста проекта.

## DECISIONS.md

Хранит историю всех важных решений проекта.

Не содержит идеи и задачи.

Используется как память проекта.

Перед принятием новых решений агент должен сверяться с этим файлом.

После принятия любого важного решения агент обязан его обновить.

Примеры решений:

- изменение структуры сайта
- изменение визуального стиля
- добавление/удаление страниц
- изменение принципов навигации
- отказ от старых идей
- утверждение новых направлений проекта
