# GIF WORLD · после MVP — план «ощущение»

Открывай этот файл, когда технический MVP (деплой, guest book, smoke test) уже закрыт.

**Цель:** гость за ~30 секунд чувствует «чей-то дом на старом интернете», а не «демо с виджетами».

**Критерий успеха:** ты открываешь main и думаешь «это мой уголок»; друг может написать в guest book и пощёлкать — и видит ответ, не инструкцию.

---

## 1. Голос автора (мало кода, большой эффект)

- [ ] **Intro на main** — твой тон (как в about): neocities, «я делаю это сейчас», без списка фич.
- [ ] **Scrollport «что уже здесь»** — убрать с главной **или** заменить на **UPDATES.log** (3–7 строк: что нового на сайте).
- [ ] **Публичный todo** (по желанию, fish2fish) — «сейчас делаю / хочу», 5 пунктов от руки, не `[x] radio`.

→ Сайт ведёт человек, не roadmap.

---

## 2. Социальность и живость (korenara + fish2fish)

- [ ] **Guest book** — Supabase настроен, на diary видны чужие записи.
- [ ] **Chatterbox / полевая лента** — короткие реплики; клик → строка в EVENTS справа.
- [ ] **Stamps / sticker wall** — 6–10 кликабельных стикеров, «липнут» локально.

→ Место общее, не музей.

---

## 3. Scrapbook внутри `.core` (korenara — точечно, не весь main)

- [ ] В **1–2 блоках** центра: бумага, линейка, наклейки, рукописный акцент (note, updates, diary-preview).
- [ ] Меньше однотипных `.exe`-лейблов — не «панель управления».
- [ ] Footer blinkies — больше своих (`wall-*`), меньше чужих архивных.

→ Коллаж и уют, поле остаётся фоном.

---

## 4. Комнаты = места (garfriend)

- [ ] **Одна комната** (gallery **или** zhanna): фон-картинка + hotspots / ощущение «зашла внутрь».
- [ ] **Gateway** — довести narrative-вход «ты пришла с поля».
- [ ] Nav — короткая фраза к каждой ссылке («здесь гифки», «тетрадь»).

→ Исследование, не меню страниц.

## Порядок работ

| Шаг | Задача | Ориентир |
|-----|--------|----------|
| **A** | Твой intro + UPDATES вместо dev-scrollport на main | 1–2 ч + твой текст |
| **B** | Guest book Supabase + проверка diary | 1–2 ч |
| **C** | Chatterbox → EVENTS | 2–4 ч |
| **D** | Scrapbook-слой в одном блоке | 2–3 ч |
| **E** | Одна «комната-картинка» | 4–8 ч |
| **F** | Deploy + пройти сайт глазами гостя | 2–3 ч |

- **A + B + F** — MVP галочка + уже теплее.
- **A + B + C + D** — близко к neocities-ощущению.
- **E** — уровень «как garfriend».
