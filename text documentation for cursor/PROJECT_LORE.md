# PROJECT LORE

GIF WORLD — это отдельный цифровой мир.

Это не обычный сайт.

Это место, существующее где-то внутри интернета.

Здесь нет единого сюжета.

Мир постепенно строится и развивается.

В нём существуют:

- комнаты;
- архивы;
- музыка;
- галереи;
- существа;
- странные системы;
- секретные страницы.

Некоторые объекты ведут себя странно.

Некоторые кнопки работают неожиданным образом.

Некоторые сообщения выглядят так, будто их оставила сама система.

Первое время, только зайдя на сайт, пользователь не сразу поймет что к чему – и это нормально. 

Не нужно объяснять пользователю всё происходящее.

Часть мира должна оставаться загадкой.
