## Как мы наполняем сайт (концепт)

Ты даёшь **объекты, атмосферу, визуал из Figma** — коробка, улитка, win98-окна, коллажи.

Я (и код) можем добавлять **сайт-native штуки**, которые в макете не были, но усиливают мир — как **SIGNAL CATCH**: мини-игра, связка с LIVE SYSTEM, своя палитра.

Не всё должно быть «как в макете 1:1». Макет = дух и пропорции; сайт = живые объекты + сюрпризы.

## Форматы микро-контента на странице

Набор паттернов, которые можно вставлять в object-worlds, win98-окна, боковые зоны и `.core`.

### Мини-зоны с прокруткой (scrollport)

Маленький **контейнер с рамкой**, внутри — свой скролл. Страница не прыгает, крутится только «окно».

Примеры содержимого:

- лента GIF / полароидов / стикеров;
- дневник или «архив находок» (короткие заметки);
- чат-лог «системы» или фейковый терминал;
- список треков / ссылок / цитат;
- горизонтальный скролл наклеек в коробке или на win98-панели.

Визуально: пунктир, inset-тень, win98-рамка, «экран монитора» — в духе уже существующих `.mini-world` и forecast.exe.

### Мини-опросики

Не корпоративные формы — **странные личные опросы** внутри объекта:

- «какой у тебя vibe поля сегодня?» (3 цветные кнопки);
- «улитка или сигнал?» → результат в EVENTS;
- один вопрос на экран, без регистрации;
- опрос, который «ломается» после странного ответа;
- накопительный опрос: ответы меняют цвет блока или фон мини-зоны.

Можно связать с правой колонкой (LIVE SYSTEM) и object-worlds.

### Цветные JS-вставки (не Java, но с духом старого веба)

Маленькие **живые, пёстрые, милые** блоки на JavaScript — не стерильные виджеты:

- мигающие/пульсирующие чипы с символами (как в игре, но отдельно — «счётчик настроения поля»);
- кнопка, которая меняет палитру только внутри своего окна;
- «конфетти» / звёздочки при клике (лёгко, без тяжёлого canvas);
- цветной счётчик кликов / дней / «сколько раз ты открыл коробку»;
- рандомный «артефакт дня» в ярком блоке;
- мини-визуализация: полоски, equalizer, fake progress bar «загрузка воспоминаний…».

Палитра — те же «секси оттенки» главной: `#e8a0c8`, `#8fd4a8`, `#9ec5e8`, `#e8c86a`, кремовый фон.

## Миры внутри объектов

Принцип проектирования (см. `PROJECT_VISION.md`, `DECISIONS.md`): предмет на странице → при клике/открытии → маленький мир внутри, не обычный попап.

### Главная (центральная комната)

- **Пунктирная коробка** — внутри GIF-галерея или «артефакт дня»; рамка может «ломаться» при частых кликах. *(реализовано: hover open PNG)*
- **Улитка** — *(реализовано: автоползание по `.core`, drag, `snail.exe`, ~320px, `images/snail.png`)*
- **forecast.exe (Win98)** — *(реализовано: Forecast + «НОВАЯ КОЛЛЕКЦИЯ», меню, Look up → попап; вселенная коллекции — позже)*
- **Конверт на столе** — письмо от «системы» или гостевая книга с глюками.
- **Монитор / маленький экран** — fake OS: терминал, счётчик сигнала, странные логи.
- **Игрушка (улитка, лягушка)** — мини-игра или пасхалка; после N кликов — секрет.
- **Книга / блокнот** — заметки, стикеры, дневник; страницы листаются внутри объекта.

### Галерея

- **Полароид на стене** — внутри одна GIF + история находки.
- **Коробка с наклейками** — коллекция мини-GIF, каждая наклейка кликабельна.
- **Витрина / ящик** — «редкие артефакты» с отдельным тёмным интерьером.

### Музыкальная комната

- **Кассета / CD-case** — плейлист, визуализация, «трек дня».
- **Радио / boombox** — переключение «каналов» (lofi, шум поля, голос дерева).
- **Наушники** — изолированный мир: один трек + анимация + текст.

### Рабочий стол

- **Папка на столе** — уже по Vision; внутри файлы, окна, ярлыки.
- **Иконка на рабочем столе** — каждая иконка = мини-приложение (блокнот, paint, error).
- **Recycle bin** — удалённые «воспоминания», иногда случайный секрет.

### Дом / отель

- **Шкаф / ящик в комнате** — другой сезон, другой фон или скрытая дверь.
- **Окно** — вид на другую локацию сайта (поле, лес, «небо с сигналом»).
- **Дверь с замком** — код, ключ из другого объекта, переход в комнату.

### Архив воспоминаний

- **Архивная коробка с номером** — год или тема; внутри подборка заметок.
- **Фотоальбом** — листание; между страницами скрытые вкладки.
- **Каталожная карточка** — одна память разворачивается в полноценный мини-архив.

### Секретные страницы и переходы

- **Предмет без подписи** — только hover «?»; внутри пустая страница или одна картинка с напутствием.
- **Сломанная кнопка** — после серии действий открывает «подпольный» мир.
- **Растение** — растёт при возвратах на сайт; на стадии N открывает внутренний «лог садовника».
